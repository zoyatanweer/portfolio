CSS Text Property

CSS (Cascading Style Sheets) is a frontend language for Web Development. In this blog, I am going to talk about various text properties.

  1. text-align

    This property is used to align the text in HTML. We have four values: left, right, center and justify. We write them as follows:

    • For text to be left-aligned : p{ text-align:left; }
    • For text to be center-aligned : p{ text-align:center; }
    • For text to be right-aligned : p{ text-align:right; }
    • For text to be justified : p{ text-align:justify; }

  2. text-decoration

    This property is used for applying decoration on a text. It has values such as overline, underline, line-through. Suppose we want our paragraph to have these effects, we write:

    • p{ text-decoration:underline; }

    • p{ text-decoration:overline; }
    • p{ text-decoration:line-through; }

  3. line-height

    This controls the height of the line. Line-height can have values in form of numbers, pixels(px), percentage, em and rem. For having line-height in our paragraph, we write:

    • line-height:1.5;
    • line-height:100px;
    • line-height:100%;
    • line-height:2rem;

  4. letter-spacing

    This property controls the leter spacing between the texts. It has values: normal, rem, em, pixels(px).

    • letter-spacing: normal;
    • letter-spacing:1rem;
    • letter-spacing:1.5em;
    • letter-spacng:10px;

  5. color

    The color property allows us to change the text color of our document. It has values in rgb, hex and names.

    • color: red;
    • color: rgb(173,34,60);
    • color: #4361ee;

  6. text-tansform

    This property is used to chage the text into lowercase, uppercase or capitalise.To capitalise the first letter of our text, we write capitalize.

    • text-transform: lowercase;
    • text-transform: uppercase;
    • text-transform: capitalize;

  7. text-shadow

    This property adds shadows to text. It has four values: horizaontal, vertical, blur radius and color. For having text shadow on our paragraph, we write:

    • text-shadow:5px 3px 2px red;

  8. font-family

    The font-family property allows us to set different fonts on our text.

    • font-family: serif;
    • font-family: sans-serif;
