CSS Selectors and Combinators

There are quite a few different ways to select elements. In this blog, I tried to discuss all of them.

CSS Selectors

  1. Universal Selector (*)

    The universal selector * selects everything in the document. If there are buttons or input on the page, then it will also have the styles applied when written in this selector. This selector is generally not used as this is not considerd as a good practice to have the same exact style all over the page. It's written as :


    * { color: blue; }

  2. Element Selector

    The element selector selects the HTML elements like <h1> , <p> , <button> , <input> etc. It basically selects everything of the type mentioned . We write it as :


    h1{ font-size: 1.5rem ; color: cyan; }

  3. Grouping Selector

    This selector is very useful selector. Suppose we want to apply same styles to two or more elements. So what we do is, instead of writing same code again and again using 'element selector' as we saw earlier, we use the 'grouping selector' to make our code short and clean.
    Remember, a nice clean code is all that you would want to see rather than seeing a clutter and not understanding a word. So here it goes. We write the grouping selector by inserting commas between the elements. Like this:


    h3, p{ color: pink ; }
    Here, both the <h3> and <p> will have the color pink.

  4. ID Selector

    This selector is used for giving CSS styles to any element having an id. In HTML, we give certain class and id to elements to make then unique and give the CSS styles as we want. When we have to define any element with an ID, we write as id="id_name" .
    If we write id in any element, then that means that that id is unique and we cannot give same id to other elements. We can only use a particular id for a single element and is used only once.


    If we have any element written as: <p id="para-id" > This is example of ID selector </p>
    then for this we'll write in CSS as : #para-id{ color: red; }

  5. Class Selector

    This selector is used to give CSS styles to any element having a class. In HTML, we give class name in any element as class="class_name" .
    When we give any element a class, then it means that we can use the same class for multiple elements and all those elements will have the same effect. For e.g : If we have code like


    <h4 class="para-class" > This is line 1 of Class Selector </h4>
    <p class="para-class" > This is line 2 of ID selector </p>
    then for this CSS would be:
    .para-class{ color: green; }

  6. Attribute Selector

    The Attribute Selector selects the elements which have some particular attribute. For example, If we want to select <input> where type attribute is password. So for it we'll write:


    input [type="password"] { width:40px; color: yellow; }

CSS Combinators

  1. Descendant Selector (space)

    This selector is written by using space. This is used to select elements which are descendants of an element.


    Suppose I have two <a> inside a <li> and we want it to have violet color. So for that, we'll write:
    li a{ color: violet ; }

  2. Adjacent Selector (+)

    The Adjacent Selector is used to select those elements which are directly under a particular element. Like if I have <p> under <h1> then for that I willwrite:


    h1 + p { color: yellow; }
    Note that here + is a combinator

  3. Direct Child Selector (>)

    Direct Child Seletor selects all the elements which are child of a specified element. For selecting a <li> under a <div> we write:


    div > li{ color: grey; }

  4. General Sibling Selector (~)

    The General Sibling Selector selects the elements which are the siblings. If we have <div> under a <p> (not necessarily immediately), then we can write:


    div ~ p{ color: cream; }
